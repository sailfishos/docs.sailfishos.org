# Hardware development

This page is about developing hardware for Jolla devices.

## The Other Half development

_Jolla Phone (2026)_ features _The Other Half (TOH)_ functionality.
TOHs are back covers that can interact with the phone software.
This is enabled via pogo pins on the back of the device that provide power, charging input, I²C / I3C communication and interrupt.
The device also features mounting points intended for heavier versions of TOH but most are expected to just clip on.

See [Jolla Phone (2026) The Other Half development](/Develop/Hardware/JP2601-TOH) for more information.
